Animal Education Website

A simple static website built using HTML, CSS, and JavaScript, designed to provide educational information about animals.
This project demonstrates fundamental front-end development concepts, including page navigation, layout styling, and basic JavaScript interaction.

The site is intentionally lightweight and does not rely on frameworks or build tools, making it easy to deploy on any static hosting platform.
Tech Stack : 
HTML5
CSS3
JavaScript (Vanilla)

No frameworks or external libraries are used in order to demonstrate core web fundamentals.
Project Outline : 

├── index.html              # Homepage
├── about.html              # Information about the project/site
├── contact.html            # Contact page
├── animalEducation.html    # Educational content page
├── main.css                # Global site styling
├── script.js               # Basic JavaScript functionality
├── images/                 # Static image assets
└── dropdown/               # Navigation dropdown components

Features

Multi-page static website

Responsive navigation bar with dropdown menu

Clean CSS styling with a single global stylesheet

Simple JavaScript for UI behavior

Organized folder structure for assets

Purpose of This Project

This project was created as a learning exercise in front-end development and to practice:

structuring multi-page websites

organizing static assets

writing maintainable CSS

Implementing simple navigation interactions with JavaScript

Possible Future Improvements
